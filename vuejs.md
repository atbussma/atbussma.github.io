# Vue.js 3

## Install

* npm install vue@next
* npm install -D @vue/compiler-sfc
* npm install -g @vue/cli

## Vite.js Application Bootstrap

* npm init @vitejs/app {application-name}
* cd {application-name}
* npm install
* npm run dev

### Configuration

* Vue
* Vue-TS
